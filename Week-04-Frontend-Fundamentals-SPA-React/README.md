# Week 4: Frontend Fundamentals, Single Page Applications & React State Management

## Topics Covered

- Frontend Fundamentals (HTML5 Semantic Elements, Accessible Web Forms, Web Storage API)
- Modern CSS3 (CSS Variables, Flexbox Layouts, 2D/3D Grid Layouts, Responsive Media Queries, Micro-animations)
- Modern ES6+ JavaScript (`let`/`const`, Arrow Functions, Destructuring, Spread/Rest Operators, Async/Await, Promises, ES Modules)
- Single Page Applications (SPA Architecture vs. Multi-Page Applications, Virtual DOM Reconciliation)
- React Component Fundamentals (JSX Syntax, Functional Components, Props vs. State, Conditional Rendering, Lists & Keys)
- React State Management (`useState`, `useEffect` & Lifecycle Simulation, `useReducer` for Complex State, Context API to eliminate Prop Drilling)

- Implemented robust state management using `useState`, `useReducer`, and the Context API across component hierarchies.
