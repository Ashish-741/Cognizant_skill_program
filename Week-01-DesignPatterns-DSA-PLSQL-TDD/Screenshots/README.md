# Screenshots & Execution Evidence

This directory is intended to store screenshot proofs of local execution for Week 1 submissions.

## Recommended Screenshots to Capture

1. **Design Patterns Execution:**
   - Screenshot of Eclipse / IntelliJ IDEA console output running `SingletonDemo.java`, `FactoryDemo.java`, and `StrategyDemo.java`.

2. **PL/SQL Script Execution:**
   - Screenshot of Oracle SQL Developer / SQL*Plus / APEX showing successful execution of `DDL.sql`, `DML.sql`, and query outputs from `Queries.sql`.
   - Proof of stored procedure `give_department_hike` execution and trigger audit log insertion.

3. **TDD / JUnit Test Reports:**
   - Screenshot of the green test bar in IntelliJ IDEA or Eclipse JUnit Runner for `BankAccountTest.java` and `OrderServiceTest.java`.

4. **Logging Output:**
   - Screenshot showing console log output formatted by SLF4J / Logback with different log levels (`INFO`, `DEBUG`, `WARN`, `ERROR`).

> **Tip:** Name your screenshot images clearly, such as `01_Singleton_Output.png`, `02_SQL_Queries.png`, `03_JUnit_GreenBar.png`, and place them directly in this folder before final evaluation.
